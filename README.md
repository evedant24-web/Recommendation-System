# Recommendation System using Collaborative Filtering

## Project Overview
This project builds a movie recommendation system using collaborative filtering techniques. It uses both memory-based (cosine similarity) and model-based (SVD) approaches.

## Features
- Long-tail distribution analysis
- Manual cosine similarity (NumPy)
- SVD-based recommendation
- RMSE evaluation
- Top 5 movie recommendations

## How to Run
1. Install libraries:
pip install pandas numpy matplotlib scikit-learn

2. Run:
python main.py

## Output
- Graph of movie popularity
- RMSE value
- Top 5 recommended movies
